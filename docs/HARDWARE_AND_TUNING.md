# 接线和调参

## CAN 分配

### CAN1：底盘

- C 板 PD0/PD1。
- 四个 C620 串接。
- 总线两端各一个 120 Ω 终端电阻。
- 电机反馈 ID 必须依次为 `0x201`、`0x202`、`0x203`、`0x204`。

### CAN2：云台

- C 板 PB5/PB6。
- 6623 和 GM6020 串接。
- 6623 拨码设为 Yaw，对应反馈 `0x205`。
- GM6020 设为 ID 2，对应反馈 `0x206`。
- 6623 供电范围和终端电阻遵循其官方手册。

## 电机方向确认

把整车架空，限制最大电流后测试：

1. 给很小的正向命令。
2. 左前 1 与左后 3 应产生相同物理方向。
3. 右前 2 与右后 4 应产生相同物理方向。
4. 左右两侧都应推动小车向前。
5. 若某个轮方向错误，只修改 `CHASSIS_MOTOR_DIRECTION` 对应项。

不要通过调换电机 ID 修方向，否则里程计和故障定位会混乱。

## 底盘 PID

从低电流开始：

1. `ki=0`，逐步增加 `kp`，直到速度响应足够快但不持续振荡。
2. 少量增加 `ki` 消除稳态误差。
3. 普通底盘通常不需要明显的 `kd`。
4. 检查堵转电流和 C620 温度。

## 云台 PID

云台采用位置外环、速度内环：

1. 先将位置环输出限幅调低。
2. 只调速度内环，使恒定速度目标平稳。
3. 再调位置外环 `kp`。
4. 必要时少量增加位置环 `kd`。
5. 6623 速度来自编码器差分，先观察噪声再调整低通系数。

俯仰轴必须先确认软件限位和机械限位方向。负载、重心和摩擦会显著影响 PID，
仓库默认参数不能替代实机标定。

## 里程计标定

1. 精确测量轮子有效滚动半径。
2. 让小车直行固定距离，修正轮半径比例。
3. 原地旋转多圈，修正等效轮距。
4. 分别校正左右侧速度比例。
5. 接入 IMU 后，用外部 yaw 抑制轮滑导致的航向漂移。
