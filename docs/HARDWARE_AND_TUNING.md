# 接线和调参

## CAN 分配

### CAN1：底盘

- C 板 PD0/PD1。
- 四个 C620 串接。
- 总线两端各一个 120 Ω 终端电阻。
- 电机反馈 ID 必须依次为 `0x201`、`0x202`、`0x203`、`0x204`。

正式整车固件要求 ID1～4 四台电机同时在线：

- 任意一台反馈超时，底盘电流整体清零并清空 PID 与软启动状态。
- 四台恢复在线并重新解锁后，从零转速、零电流开始软启动。
- `CHASSIS_MOTOR_ONLINE_MASK` 的 bit0～bit3 对应 ID1～ID4。

四台同时接入出现异常时，通过 SWD 查看：

- `CAN1_ID_RX_COUNT[0..4]`：ID1～ID4 各自的反馈帧数。
- `CAN1_ERROR_STATUS`：STM32 CAN1 的原始 ESR。
- `CAN1_TX_ERROR_COUNT`：发送错误计数 TEC。
- `CAN1_RX_ERROR_COUNT`：接收错误计数 REC。

如果 `CAN1_RX_COUNT` 为零且 TEC 持续升高，问题位于 CAN 物理层或节点供电，
不是电机在线策略。断电后测量 CAN_H 与 CAN_L：总线两端各一个 120 Ω 终端时，
等效电阻应接近 60 Ω。逐台接入可定位反接、短路、额外终端或损坏线束。

### CAN2：云台

- C 板 PB5/PB6。
- 6623 和 GM6020 串接。
- 6623 拨码设为 Yaw，对应反馈 `0x205`。
- GM6020 设为 ID 2，对应反馈 `0x206`。
- 6623 供电范围和终端电阻遵循其官方手册。

## 电机方向确认

把整车架空，限制最大电流后测试：

1. 给很小的正向命令。
2. 左前 1 与左后 3 应产生相同物理方向。
3. 右前 2 与右后 4 应产生相同物理方向。
4. 左右两侧都应推动小车向前。
5. 若某个轮方向错误，只修改 `CHASSIS_MOTOR_DIRECTION` 对应项。

不要通过调换电机 ID 修方向，否则里程计和故障定位会混乱。

## 轮胎模式与 SwB

FS-i6 的辅助通道设置：

- CH6：`SwB`
- CH5：`SwC`（本机安全开关配置）

`SwB` 上档选择普通轮模式，下档选择麦克纳姆轮模式。可通过 SWD 查看：

- `SWB_CHANNEL_RAW`：SwB 原始值，上档通常约 `-784`，下档约 `+783`。
- `CHASSIS_WHEEL_MODE`：`0` 表示普通轮，`1` 表示麦克纳姆轮。

麦克纳姆轮必须按 X 型辊子方向安装。架空测试顺序：

1. SwB 上档，左摇杆前推，四轮应共同产生向前运动。
2. SwB 上档，左摇杆横推，左右两侧应产生相反方向用于转向。
3. SwB 下档，左摇杆前推，四轮仍共同产生向前运动。
4. SwB 下档，左摇杆右推，逻辑轮速应为左前正、右前负、左后负、右后正。
5. 切换 SwB 的瞬间，四路目标电流应有一个周期为零。

如果前后方向整体错误，修改 `CHASSIS_MOTOR_DIRECTION`；如果只有横移方向错误，
先检查四个麦克纳姆轮是否装反，不要用交换电机 ID 的方式修正。

## 底盘 PID

从低电流开始：

1. 默认单路电流限幅 `6000`，四路总预算 `12000`。
2. 默认转速斜坡 `2000 RPM/s`，电流斜坡 `20000/s`。
3. 先确认四台同时启动时母线电压稳定，再逐步提高总电流预算。
4. `ki=0`，逐步增加 `kp`，直到速度响应足够快但不持续振荡。
5. 少量增加 `ki` 消除稳态误差。
6. 普通底盘通常不需要明显的 `kd`。
7. 检查堵转电流和 C620 温度。

## 云台 PID

云台采用位置外环、速度内环：

1. 在 `src/config/gimbal.rs` 填写俯仰编码器机械零点和方向。
2. 架空确认绝对角度与机械方向一致后，将 `GIMBAL_CALIBRATION.calibrated` 改为 `true`。
3. 先将位置环输出限幅调低。
4. 只调速度内环，使恒定速度目标平稳。
5. 再调位置外环 `kp`。
6. 必要时少量增加位置环 `kd`。
7. 6623 速度来自带时间戳的编码器差分，先观察噪声再调整低通系数。

俯仰轴必须先确认软件限位和机械限位方向。负载、重心和摩擦会显著影响 PID，
仓库默认参数不能替代实机标定。默认配置未标定，云台不会输出电流。

## 里程计标定

1. 精确测量轮子有效滚动半径。
2. 让小车直行固定距离，修正轮半径比例。
3. 原地旋转多圈，修正等效轮距。
4. 分别校正左右侧速度比例。
5. 接入 IMU 后，用外部 yaw 抑制轮滑导致的航向漂移。
